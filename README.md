## ⚙️ Admin Panel
- Click **⚙ ADMIN** button (top-right) to enter admin mode
- Edit buttons appear on each section
- All changes saved to browser localStorage automatically

## 🔧 Customization
- Open `index.html` in any text editor
- Find `DEFAULT_DATA` object in `<script>` section
- Edit values directly for permanent changes
